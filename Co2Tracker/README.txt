# Co2Tracker - Installazione su Google Chrome

## Descrizione
Co2Tracker è un'estensione per Google Chrome progettata per monitorare e ridurre la tua impronta di carbonio durante la navigazione web.

## Requisiti
- Google Chrome installato sul tuo computer
- File dell'estensione Co2Tracker (in formato .zip o cartella sorgente)

## Installazione
### Metodo 1: Installazione manuale da file sorgente
1. **Scaricare i file dell'estensione**
   - Scarica l'ultima versione di Co2Tracker dal nostro sito ufficiale o dal repository GitHub.
   - Se hai un file .zip, estrailo in una cartella.

2. **Aprire la pagina delle Estensioni di Chrome**
   - Apri Google Chrome
   - Digita nella barra degli indirizzi: `chrome://extensions/` e premi Invio

3. **Abilitare la Modalità Sviluppatore**
   - In alto a destra, attiva l'interruttore "Modalità sviluppatore".

4. **Caricare l'estensione**
   - Clicca su "Carica estensione non pacchettizzata"
   - Seleziona la cartella in cui hai estratto Co2Tracker
   - Premi "Seleziona cartella"

5. **Verificare l'installazione**
   - L'estensione verrà caricata e comparirà nell'elenco delle estensioni attive
   - Clicca sull'icona di Co2Tracker nella barra delle estensioni per avviarla

### Metodo 2: Installazione da Chrome Web Store (se disponibile)
1. **Visitare il Chrome Web Store**
   - Apri Chrome e vai su [Chrome Web Store](https://chrome.google.com/webstore/)
   
2. **Cercare Co2Tracker**
   - Usa la barra di ricerca e digita "Co2Tracker"
   
3. **Installare l'estensione**
   - Clicca su "Aggiungi a Chrome"
   - Conferma cliccando su "Aggiungi estensione"

4. **Verificare l'installazione**
   - L'icona di Co2Tracker dovrebbe apparire nella barra delle estensioni
   - Cliccala per iniziare a monitorare la tua impronta di carbonio!

## Disinstallazione
- Vai su `chrome://extensions/`
- Trova Co2Tracker
- Clicca su "Rimuovi"
- Conferma la rimozione
